### W- Question ❔

| German    | English  |
| --------- | -------- |
| Wo        | Where    |
| Was       | What     |
| Wann      | When     |
| Warum     | Why      |
| Wie       | How      |
| Wer       | Who      |
| Wie viel  | How much |
| Wie Viele | How many |
| Wessen    | Whose    |

### Personal Pronoun

| German    | English   |
| --------- | --------- |
| ich       | I         |
| du        | you       |
| er/sie/es | he/she/it |
| wir       | we        |
| ihr       | you       |
| Sie       | they      |
